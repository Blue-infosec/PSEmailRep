# Set-EmailRep

## SYNOPSIS
Set API Key for persistent use.

## SYNTAX

```
Set-EmailRep [-APIKey] [-Test] [<CommonParameters>]
```

## DESCRIPTION
Set API Key for persistent use in encrypted string.
Default file location is in user's AppData folder.

## EXAMPLES

### EXAMPLE 1
```
Set-EmailRep -API
```

API Key: ***************************************
